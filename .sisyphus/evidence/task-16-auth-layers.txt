## Architecture Notes
- Dependency inversion: CapabilityLookup interface (not direct @aoa/kya import)
- ValidationSchema interface compatible with Zod safeParse (no zod dependency)
- Only dependency added: hono@4.12.7
- In-memory rate-limit store (Map<IP, timestamps[]>) with sliding window cleanup
- PII masking is log-only — never mutates actual request/response body
- Routes: 6 endpoints (1 public, 3 auth-only, 2 auth+capability)
